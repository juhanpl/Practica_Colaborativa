JavaScript

JavaScript es un lenguaje de programación interpretado que permite crear interactividad en páginas web. Es ampliamente utilizado en el desarrollo frontend, aunque también se emplea en el backend con entornos como Node.js.

Es uno de los tres pilares fundamentales del desarrollo web junto con HTML y CSS. Permite validar formularios, crear animaciones, actualizar contenido dinámicamente sin recargar la página (AJAX), y más.

Características clave:

  • Lenguaje interpretado del lado del cliente
  • Basado en eventos y orientado a objetos
  • Funciona en todos los navegadores modernos
  • Soporte para programación asincrónica con async / await

PHP

PHP (Hypertext Preprocessor) es un lenguaje de programación del lado del servidor, diseñado para el desarrollo web. Es ampliamente utilizado para crear sitios web dinámicos y aplicaciones que interactúan con bases de datos.

Fue uno de los primeros lenguajes del lado del servidor que se podían incrustar directamente en HTML. Hoy en día sigue siendo una tecnología clave en plataformas como WordPress, Joomla, Drupal, Laravel, entre otras.

Características clave:

  • Lenguaje de servidor embebido en HTML
  • Compatible con múltiples bases de datos (MySQL, PostgreSQL, SQLite...)
  • Gran comunidad y abundante documentación
  • Ideal para proyectos CMS y eCommerce

Ejemplo de código PHP

<?php
            $nombre = "Mundo";
            echo "¡Hola, " . $nombre . "!";
            ?>

Lenguaje Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Es muy utilizado en desarrollo web, ciencia de datos, inteligencia artificial, automatización y más.

Características principales:

  • Sintaxis simple y clara
  • Gran cantidad de bibliotecas y frameworks
  • Multiparadigma (soporta programación orientada a objetos, funcional, etc.)

Ejemplo de código en Python:


# Programa que solicita al usuario su nombre
nombre = input("¿Cómo te llamas? ")
print(f"¡Hola, {nombre}!")

Lenguaje Java

Java es un lenguaje de programación de propósito general, orientado a objetos y ampliamente utilizado en aplicaciones empresariales, móviles (Android) y de escritorio.

Características principales:

  • Lenguaje compilado y fuertemente tipado
  • Orientado a objetos
  • Gran portabilidad: "Escribe una vez, ejecuta en cualquier lugar"

Ejemplo de código en Java:


// Programa que solicita el nombre del usuario
import java.util.Scanner;

public class HolaMundo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("¿Cómo te llamas? ");
        String nombre = sc.nextLine();
        System.out.println("¡Hola, " + nombre + "!");
        sc.close();
    }
}
